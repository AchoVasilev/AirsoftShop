# AirsoftShop
Small .NET and Angular project

### The project uses:
- MS SQL database
- Entityframework Core
- ASP.NET Core Web API for back-end
- Cloudinary for image storage
- Automapper
- Font awesome
- NewtonSoft Json
- Angular
- Angular Material
- Toaster
- TypeScript
- Custom written HTML and CSS

### Features
## Not logged in user
- Access index page for searching
- View the list with all guns
- See gun details

## Registered client
- Access index page
- View the list with all guns
- Make orders
- See gun details
- Change client data

## Registered company
- Can add, edit and delete their guns
- Access index page
- View the list with all guns
- See gun details
